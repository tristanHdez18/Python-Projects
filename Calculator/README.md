# Calulator

### A basic calculator app that takes two numbers as input, then performs *one of* addition, multiplication, subtraction, division.

### To run

You need to put the next command to run background:

For **Windows**:
```
pythonw .\main.py
```

For **Linux**:
```
python3 main.py&
```

or run:
```
python3 main.py
```
